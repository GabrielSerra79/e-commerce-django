# e-commerce-django
Repo de estudo para o Curso de Python 3 do básico ao avançado - com projetos reais (Prof. Luiz Otávio Miranda) na Udemy

# Refertencias
* [Django Debug Toolbar](https://django-debug-toolbar.readthedocs.io/en/latest/)

<br>

# Criar ambiente virtual
```bash
python3 -m venv venv
```

Ativar o ambiente virtual
```bash
. venv/bin/activate
```

Istalar dependencias
```bash
pip install -r requirements.txt
```

Fazer a migraçao para o Django
```bash
python manage.py migrate
```

Criar superuser no Django
```bash
python manage.py createsuperuser
```

Rodar o projeto
```bash
python manage.py runserver
```
